## 3. 荷载分项安全系数 (Section 7.4, Table 3)

| 设计情况类型 | 不利荷载 γf | 有利荷载 γf |
|-------------|-----------|-----------|
| 正常 (N) | **1.35*** | **0.9** |
| 异常 (A) | **1.1** | 0.9 |
| 运输/安装 (T) | **1.5** | 0.9 |

> *DLC 1.1（极值统计外推法）可用 γf = 1.25

### 重力荷载分项公式

当重力为不利荷载且与其他荷载组合时：

```
γf = 
  if Fgravity_k ≤ 0: γf = 1.0 (有利)
  if Fgravity_k > 0: γf = γf_gravity = 1.35 | 1.1 | 1.5 (取决于设计情况)
```

## 4. 材料分项安全系数 (Section 7.6.2)

### 钢材

| 失效类型 | γm |
|---------|-----|
| 稳定性丧失（屈曲） | 1.1 |
| 极限强度（断裂） | 1.2 |

### 极限强度校核公式

```
Fd = γf * Fk     (设计荷载 = 分项系数 × 特征荷载)
Rd = Rk / γm     (设计抗力 = 特征抗力 / 材料系数)
Fd ≤ Rd          (设计荷载 ≤ 设计抗力)
```

## 5. 疲劳校核 (Section 7.6.3)

- **方法**：Palmgren-Miner 线性累积损伤准则
- **损伤累积**：D = Σ(ni / Ni) < 1.0
  - ni = 实际循环次数（应力范围 Si 下）
  - Ni = S-N 曲线在应力范围 Si 下的容许循环次数
- **等效疲劳荷载法**(DEL)：将变幅荷载简化为等效应力范围
- **疲劳安全系数**：γf = 1.0（对疲劳荷载），γm 取决于材料和细节类型

## 6. 荷载类型 (Section 7.3)

### 7.3.1 重力荷载
- 各部件自重（叶片、轮毂、机舱、塔筒、基础）
- 恒载分项系数见 Table 3

### 7.3.2 惯性和回转效应
- 转子转动产生的离心力
- 陀螺力矩（偏航时）
- 制动产生的惯性力

### 7.3.3 气动荷载
- 风与叶片/塔筒相互作用产生的气动力
- 包括：升力、阻力、力矩
- 考虑湍流、风切变、阵风效应

### 7.3.4 其他荷载
- 波浪荷载（海上风电）
- 冰荷载
- 地震荷载
- 交通荷载

## 7. 关键计算公式

### 7.1 相干阵风幅值 (EOG)

```
Vgust_N = β * [1 - cos(2πt/T)] / 2

其中：
β = 阵风幅值 (m/s)，取决于重现期和安全等级
T = 阵风周期（通常 10.5s）
```

### 7.2 极端风切变 (EWS)

```
V(z,t) = V(z) * [1 ± 0.5 * (Λ/Λ0) * cos(2πt/T)]
```

### 7.3 湍流强度

```
I = σ1 / Vhub

其中：
σ1 = 纵向湍流标准差
Vhub = 轮毂高度10min平均风速
```

### 7.4 极端风速模型 (EWM)

**稳态 EWM**：
```
V50(z) = Vref * (z / zhub)^α      (50年重现期)
V1(z) = 0.8 * V50(z)               (1年重现期)
```

**湍流 EWM**：稳态风速 + 正常湍流

## 8. 设计流程 (Section 7.1 - 7.6)

```
① 确定风电机组等级 (Table 1)
② 选择设计工况 DLC (Table 2)
③ 确定荷载分项安全系数 (Table 3)
④ 执行荷载计算：
   ├─ 发电工况 → 气弹仿真 (OpenFAST)
   ├─ 极端风况 → 解析公式 + 仿真
   └─ 故障工况 → 事件仿真
⑤ 极限强度校核 (7.6.2):
   Fd = γf × Fk ≤ Rd = Rk / γm
⑥ 疲劳校核 (7.6.3):
   D = Σ(ni/Ni) < 1.0
⑦ 服役极限状态校核 (7.6.4):
   位移、变形在容许范围内
```
